Política de Privacidade

AbasteCe — Plataforma colaborativa de preços de combustível

Versão 1.0 — 17 de maio de 2025

🔒 Em conformidade com a LGPD (Lei nº 13.709/2018)

1. Introdução

Paulo César Rezende da Silva, desenvolvedor do aplicativo AbasteCe, com endereço em Mangaratiba, Rio de Janeiro, Brasil, está comprometido com a proteção dos dados pessoais dos usuários, em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018 — LGPD).

Esta Política descreve quais dados coletamos, como os utilizamos, com quem os compartilhamos e quais são seus direitos como titular dos dados.

2. Dados Coletados

O AbasteCe coleta os seguintes dados pessoais:

  • Nome de usuário — fornecido no cadastro;
  • Endereço de e-mail — utilizado para autenticação e comunicação;
  • Localização geográfica (GPS) — coletada durante o uso do aplicativo para exibir postos próximos e registrar reportes;
  • Histórico de reportes e avaliações — conteúdo gerado pelo usuário na plataforma;
  • Token de notificação push — para envio de notificações sobre preços e confirmações;
  • Data e hora de uso — para fins de segurança e moderação.

3. Finalidade do Tratamento

Os dados coletados são utilizados para:

  • Autenticar e identificar o usuário na plataforma;
  • Exibir postos de combustível próximos à localização do usuário;
  • Registrar e exibir reportes de preços inseridos pelo usuário;
  • Calcular pontos de reputação e níveis do usuário;
  • Enviar notificações sobre quedas de preço, confirmações e novos reportes;
  • Moderar conteúdo e aplicar medidas contra uso indevido da plataforma;
  • Melhorar a experiência e funcionalidades do aplicativo.

4. Base Legal

O tratamento dos dados pessoais é realizado com base nas seguintes hipóteses previstas na LGPD:

  • Consentimento do titular — ao aceitar estes termos no momento do cadastro;
  • Execução de contrato — para prestação dos serviços da plataforma;
  • Legítimo interesse — para fins de segurança, moderação e melhoria do serviço.

5. Compartilhamento de Dados

Os dados dos usuários podem ser compartilhados com os seguintes terceiros, exclusivamente para fins operacionais:

  • Supabase (Supabase Inc.) — plataforma de banco de dados e autenticação. Os dados são armazenados em servidores seguros da AWS;
  • Google LLC — para serviços de mapas (Google Maps) e notificações push (Firebase Cloud Messaging);
  • Expo (Expo Technology Inc.) — para distribuição do aplicativo e serviços de notificação push.
Não vendemos, alugamos ou compartilhamos dados pessoais com terceiros para fins comerciais ou publicitários.

6. Localização Geográfica

O aplicativo solicita acesso à localização GPS do dispositivo para funcionar corretamente. A localização é utilizada exclusivamente para:

  • Exibir postos de combustível próximos ao usuário;
  • Registrar a posição geográfica dos reportes de preço inseridos.

O usuário pode revogar a permissão de localização nas configurações do dispositivo, porém isso impedirá o funcionamento de funcionalidades essenciais do aplicativo.

7. Retenção dos Dados

Os dados pessoais são mantidos enquanto a conta do usuário estiver ativa. Após a exclusão da conta, os dados pessoais identificáveis serão removidos em até 30 (trinta) dias, ressalvados os dados cuja retenção seja exigida por obrigação legal.

Reportes e avaliações podem ser mantidos de forma anonimizada para fins estatísticos.

8. Direitos do Titular

Nos termos da LGPD, o usuário possui os seguintes direitos em relação aos seus dados pessoais:

  • Confirmação da existência de tratamento;
  • Acesso aos dados;
  • Correção de dados incompletos, inexatos ou desatualizados;
  • Anonimização, bloqueio ou eliminação de dados desnecessários;
  • Portabilidade dos dados;
  • Eliminação dos dados tratados com base no consentimento;
  • Informação sobre compartilhamento com terceiros;
  • Revogação do consentimento.

Para exercer qualquer um desses direitos, entre em contato pelo e-mail: pcrsengmec@gmail.com

9. Segurança dos Dados

Adotamos medidas técnicas e organizacionais adequadas para proteger os dados pessoais contra acesso não autorizado, perda, alteração ou divulgação indevida, incluindo:

  • Autenticação segura via Supabase Auth;
  • Políticas de controle de acesso (Row Level Security) no banco de dados;
  • Transmissão de dados criptografada via HTTPS/TLS.

10. Exclusão de Conta

O usuário pode solicitar a exclusão de sua conta e dados pessoais a qualquer momento, enviando e-mail para pcrsengmec@gmail.com com o assunto "Exclusão de conta — AbasteCe". O processamento será realizado em até 30 dias.

11. Menores de Idade

O AbasteCe não é destinado a menores de 13 (treze) anos. Não coletamos intencionalmente dados de crianças. Caso identifiquemos que dados de menores foram coletados sem consentimento parental, procederemos com a exclusão imediata.

12. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada periodicamente. O usuário será notificado sobre alterações relevantes pelo aplicativo. O uso continuado após a publicação de nova versão implica na aceitação das mudanças.
